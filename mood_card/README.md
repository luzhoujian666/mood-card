# MOOD_CARD: 오늘의 기분 처방 카드

> 오늘의 기분을 짧게 적고, 나에게 필요한 위로 문장을 카드로 남기는 감정 기록 웹앱입니다.

## 프로젝트 소개

**MOOD_CARD**는 사용자가 자신의 감정을 가볍게 기록하고, 스스로에게 해주고 싶은 말을 카드 형태로 정리할 수 있도록 만든 작은 웹 메모장입니다.

공부, 인간관계, 매일의 일과 속에서 지치거나 흔들릴 때가 있습니다. 그럴 때 긴 일기를 쓰는 것은 부담스러울 수 있습니다. 이 앱은 사용자가 오늘의 기분을 고르고, 필요한 마음 처방을 선택한 뒤, 오늘 있었던 일과 나에게 해주고 싶은 말을 짧게 적도록 도와줍니다.

입력한 내용을 바탕으로 **오늘의 기분 처방 카드**가 만들어지고, 사용자는 마음에 드는 카드를 저장할 수 있습니다.  
이 앱은 전문 상담 도구가 아니라, 하루의 감정을 조용히 정리해 보는 개인 기록장입니다.

## 주요 기능

### 오늘의 기분 선택

사용자는 현재 자신의 기분을 선택할 수 있습니다.

- 지침
- 불안함
- 속상함
- 멍함
- 기분 좋음
- 혼자 있고 싶음

선택한 기분은 이후 생성되는 카드에 반영됩니다.

### 필요한 마음 처방 선택

사용자는 지금 자신에게 필요한 위로의 방향을 선택할 수 있습니다.

- 짧은 위로
- 현실적인 조언
- 쉬어가기
- 작은 행동 추천
- 응원 한마디

예를 들어 “짧은 위로”를 선택하면 부드럽고 따뜻한 문장이, “작은 행동 추천”을 선택하면 바로 실천할 수 있는 간단한 행동이 중심이 됩니다.

### 나의 일상 한마디 적기

사용자는 세 가지 짧은 문장을 입력할 수 있습니다.

- 오늘 있었던 일 한 줄
- 나에게 가만히 해주고 싶은 말
- 오늘 실천하고 싶은 아주 작은 행동 하나

긴 일기 대신 짧은 문장으로 감정을 정리할 수 있도록 구성했습니다.

### 오늘의 기분 처방 카드 만들기

입력한 내용과 선택한 기분을 바탕으로 **오늘의 기분 처방 카드**가 생성됩니다.  
카드에는 오늘의 기분, 필요한 마음 처방, 사용자가 적은 기록, 작은 행동, 위로 문장이 함께 표시됩니다.

### 오늘의 랜덤 한마디

화면에는 **오늘의 랜덤 한마디** 영역이 있습니다.  
사용자는 버튼을 눌러 짧은 문장을 바꿔 볼 수 있습니다.

예시 문장:

- 작은 정리 하나도 충분한 시작이에요.
- 오늘의 속도는 오늘의 나에게 맞추면 돼요.
- 천천히 해도 괜찮아요.

### 내가 저장한 기분 카드

생성한 카드는 저장할 수 있습니다.  
저장된 카드는 아래쪽 **내가 저장한 기분 카드** 영역에서 다시 확인할 수 있습니다.  
필요 없는 카드는 삭제할 수 있습니다.

### 예시 카드 불러오기

처음 사용하는 사람도 앱의 흐름을 이해할 수 있도록 **기분 카드 예시 3종 불러오기** 버튼을 넣었습니다.

예시 상황:

- 시험 전날 불안한 마음
- 아무것도 하기 싫은 피곤한 날
- 친구에게 서운했던 날

## 사용 방법

1. 웹페이지를 엽니다.
2. 오늘의 기분을 선택합니다.
3. 지금 필요한 마음 처방을 선택합니다.
4. 오늘 있었던 일, 나에게 해주고 싶은 말, 작은 행동을 입력합니다.
5. **오늘의 기분 처방 카드 만들기** 버튼을 누릅니다.
6. 생성된 카드를 확인합니다.
7. 마음에 드는 카드는 저장합니다.
8. 저장한 카드는 아래 목록에서 다시 볼 수 있습니다.

## 입력 / 출력

| 구분           | 입력                                  | 출력                        |
| :------------- | :------------------------------------ | :-------------------------- |
| 기분 선택      | 지침, 불안함, 속상함, 멍함 등         | 카드에 오늘의 기분으로 표시 |
| 처방 선택      | 짧은 위로, 현실적인 조언, 쉬어가기 등 | 카드의 위로 방향 결정       |
| 일상 기록      | 오늘 있었던 일 한 줄                  | 카드 본문에 표시            |
| 나에게 하는 말 | 스스로에게 해주고 싶은 문장           | 카드에 개인 문장으로 표시   |
| 작은 행동      | 오늘 할 수 있는 작은 행동             | 카드의 실천 항목으로 표시   |
| 랜덤 한마디    | 바꾸기 버튼 클릭                      | 짧은 위로 문장 변경         |
| 카드 저장      | 저장 버튼 클릭                        | 저장한 카드 목록에 추가     |

## 프로젝트에서 신경 쓴 부분

이 앱은 사용자가 부담 없이 감정을 기록할 수 있도록 만드는 데 초점을 두었습니다.  
그래서 화면을 복잡하게 만들기보다, 기분 선택 → 마음 처방 선택 → 짧은 기록 → 카드 생성의 흐름이 자연스럽게 이어지도록 구성했습니다.

또한 감정 기록 앱이라는 주제 때문에 너무 무겁거나 전문적인 표현을 사용하지 않도록 주의했습니다.  
“치료”나 “상담”을 대신하는 앱이 아니라, 오늘의 마음을 가볍게 적어 보는 메모장이라는 점을 화면 하단에 안내했습니다.

## 아직 부족한 점

이 프로젝트를 만들고 테스트하면서 다음과 같은 한계도 발견했습니다.

### 1. 전문 상담 기능은 제공하지 않음

이 앱은 감정을 기록하고 스스로를 돌아보기 위한 메모장입니다.  
사용자의 심리 상태를 진단하거나 전문적인 상담을 제공하지는 않습니다.  
따라서 힘든 상태가 오래 지속된다면 실제 상담 기관이나 주변 사람의 도움을 받는 것이 필요합니다.

### 2. 입력한 내용을 자동으로 깊게 분석하지는 않음

사용자가 적은 문장을 복잡하게 분석하는 기능은 없습니다.  
카드는 선택한 기분과 처방 종류를 바탕으로 만들어지며, 사용자가 직접 적은 문장을 중심으로 보여 줍니다.

### 3. 저장 데이터는 사용하는 브라우저에 의존함

저장한 카드는 사용 중인 브라우저 안에서 확인할 수 있습니다.  
브라우저 데이터를 삭제하거나 다른 기기에서 접속하면 이전 기록이 보이지 않을 수 있습니다.

### 4. 긴 글보다는 짧은 기록에 적합함

이 앱은 긴 일기장보다는 짧은 감정 카드에 가깝습니다.  
따라서 자세한 하루 기록이나 긴 상담 메모를 작성하는 용도에는 적합하지 않을 수 있습니다.
